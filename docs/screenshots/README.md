# Screenshots

This directory contains screenshots for the README.

## Required Screenshots

1. **landing.png** - Landing page with hero section
2. **dashboard.png** - Dashboard view (any tier)
3. **pricing.png** - Pricing page with subscription plans

## Adding Screenshots

1. Take screenshots at 1600x900 resolution or similar
2. Optimize images using tools like TinyPNG
3. Name files according to the convention above
4. Update README.md if adding new screenshots

## Tools for Screenshot Optimization

- [TinyPNG](https://tinypng.com/) - Image compression
- [Carbon](https://carbon.now.sh/) - Code screenshots
- [Screely](https://www.screely.com/) - Beautiful browser mockups
